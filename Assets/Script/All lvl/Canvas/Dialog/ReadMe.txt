Описание работы скриптов для Диалога НПС.

Для работы с диалогом необходимо:

DialogueManager - вешать на пустой объект на сцене, отвечает за работу окна диалога. 

InstantiateDialogue -  вешать на пустой объект на сцене вместе с DialogueManager, отвечает за чтение и обрабутку xml файла. 

Метод StartDialogue() вызывается на UI-кнопке "Начать говорить".

DialogueTrigger - вешать на триггер с нпс, который будет говорить. 
Он вызывает UI-кнопку для начала диалога, также он отвечает за хранение и передачу xml файла в InstantiateDialogue.
